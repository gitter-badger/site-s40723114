GIT 基本指令

cd..  進入文件資料夾

git add .  暫存

git commit -m "  "  把暫存區提到git分支上

git status 查看git目前狀態

git log 查看提交的日誌

git clone http://...  將遠端網站clone下來

git branch 查看當前分支

git branch name 創建分支

git checkout name 切換分支

git push 將近端所儲存的東西推上遠端

python   wsgi.py    啟動網站

git pull  合併所有

git log -- oneline -- graph -- all  查看日誌當前所提交的唯一行圖形

git version 檢查版本
